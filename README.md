# Telegram Shop Bot 🤖

Telegram-бот для приема заказов через Telegram.

## Возможности

✅ Главное меню  
✅ Каталог товаров  
✅ Inline-кнопки  
✅ Оформление заказа  
✅ Уведомление о заказе  

## Технологии

- Python
- pyTelegramBotAPI

## Установка

Клонировать проект:

```bash
git clone https://github.com/ТВОЙ_НИК/telegram-shop-bot.git
Установить библиотеки:
pip install -r requirements.txt
Запуск:
python bot.py
